# Drizzle ORM — Specification

> Implementation spec for the database layer of DevRoast using Drizzle ORM + PostgreSQL + Docker Compose.

## Context

DevRoast is a code quality analyzer that gives a score from 0 to 10. The user pastes a code snippet, optionally enables "roast mode", and receives:

- **Score** (0-10, decimal)
- **Verdict** (enum: `needs_serious_help`, `rough_around_edges`, `decent_code`, `solid_work`, `exceptional`)
- **Roast quote** (sarcastic AI-generated phrase)
- **Analysis items** (list of findings with severity `critical`, `warning`, or `good`)
- **Suggested fix** (diff from original code to improved version)

Submissions are **anonymous** — no authentication. Every submission automatically enters the **shame leaderboard** (no opt-in required).

AI will be integrated via the **Vercel AI SDK** (provider-agnostic).

## Design Decisions

### Automatic Casing

We use `casing: "snake_case"` in both `drizzle.config.ts` (Drizzle Kit) and the `drizzle()` client. This allows the schema to use camelCase in TypeScript without explicit aliases — Drizzle automatically converts to snake_case in SQL.

```typescript
// Schema: camelCase without alias
lineCount: integer().notNull();
// Generated SQL: line_count integer NOT NULL
```

### No Drizzle Relations

We do not use Drizzle's `relations()`. Queries are written with `select().from().leftJoin()` — more explicit and without dependency on the `db.query` API.

### Minimal Indexes

Only one custom index: `roasts_score_idx` for the leaderboard ORDER BY. PKs already have implicit indexes. Inline FKs do not justify dedicated indexes at the expected volume.

## Tables

### `roasts`

Main table. Each row is an analyzed code submission.

| Column          | Type                      | Description                                         |
| --------------- | ------------------------- | --------------------------------------------------- |
| `id`            | `uuid` PK default random  | Unique roast identifier                             |
| `code`          | `text` NOT NULL           | Source code submitted by the user                   |
| `language`      | `varchar(50)` NOT NULL    | Detected or provided language (e.g.: `javascript`)  |
| `line_count`    | `integer` NOT NULL        | Number of lines in the code                         |
| `roast_mode`    | `boolean` default `false` | Whether the user enabled sarcasm mode               |
| `score`         | `real` NOT NULL           | Score from 0 to 10 (e.g.: `3.5`)                   |
| `verdict`       | `verdict` NOT NULL        | Verdict enum based on the score                     |
| `roast_quote`   | `text`                    | Sarcastic phrase generated by AI                    |
| `suggested_fix` | `text`                    | Improved code (diff/full version)                   |
| `created_at`    | `timestamp with tz`       | Creation timestamp, default `now()`                 |

```typescript
export const roasts = pgTable(
  "roasts",
  {
    id: uuid().defaultRandom().primaryKey(),
    code: text().notNull(),
    language: varchar({ length: 50 }).notNull(),
    lineCount: integer().notNull(),
    roastMode: boolean().default(false).notNull(),
    score: real().notNull(),
    verdict: verdictEnum().notNull(),
    roastQuote: text(),
    suggestedFix: text(),
    createdAt: timestamp({ withTimezone: true }).defaultNow().notNull(),
  },
  (table) => [index("roasts_score_idx").on(table.score)]
);
```

### `analysis_items`

Each finding from the detailed analysis. N:1 relationship with `roasts`.

| Column        | Type                     | Description                                  |
| ------------- | ------------------------ | -------------------------------------------- |
| `id`          | `uuid` PK default random | Unique item identifier                       |
| `roast_id`    | `uuid` FK -> roasts.id   | Roast this item belongs to                   |
| `severity`    | `severity` NOT NULL      | Severity level (critical/warning/good)       |
| `title`       | `varchar(200)` NOT NULL  | Short finding title                          |
| `description` | `text` NOT NULL          | Detailed explanation                         |
| `order`       | `integer` NOT NULL       | Display order (0-indexed)                    |

```typescript
export const analysisItems = pgTable("analysis_items", {
  id: uuid().defaultRandom().primaryKey(),
  roastId: uuid()
    .references(() => roasts.id, { onDelete: "cascade" })
    .notNull(),
  severity: severityEnum().notNull(),
  title: varchar({ length: 200 }).notNull(),
  description: text().notNull(),
  order: integer().notNull(),
});
```

```typescript
import { defineConfig } from "drizzle-kit";

config({ path: ".env.local" });

const databaseUrl = process.env.DATABASE_URL;

if (!databaseUrl) {
  throw new Error("DATABASE_URL is not set");
}

export default defineConfig({
  out: "./drizzle",
  schema: "./src/db/schema.ts",
  dialect: "postgresql",
  casing: "snake_case",
  dbCredentials: {
    url: databaseUrl,
  },
});
```

```typescript
import { drizzle } from "drizzle-orm/node-postgres";

const databaseUrl = process.env.DATABASE_URL;

if (!databaseUrl) {
  throw new Error("DATABASE_URL is not set");
}

export const db = drizzle(databaseUrl, {
  casing: "snake_case",
});
```

## Expected Queries

### Leaderboard (shame ranking)

```typescript
import { asc } from "drizzle-orm";

const leaderboard = await db
  .select()
  .from(roasts)
  .orderBy(asc(roasts.score))
  .limit(20);
```

### Full Roast by ID (result page)

```typescript
import { eq } from "drizzle-orm";

const [roast] = await db.select().from(roasts).where(eq(roasts.id, roastId));

const items = await db
  .select()
  .from(analysisItems)
  .where(eq(analysisItems.roastId, roastId))
  .orderBy(asc(analysisItems.order));
```

### Global Stats (footer)

```typescript
import { avg, count } from "drizzle-orm";

const [stats] = await db
  .select({
    totalRoasts: count(),
    avgScore: avg(roasts.score),
  })
  .from(roasts);
```

### Insert New Roast (after AI response)

```typescript
await db.transaction(async (tx) => {
  const [roast] = await tx
    .insert(roasts)
    .values({
      code,
      language,
      lineCount,
      roastMode,
      score,
      verdict,
      roastQuote,
      suggestedFix,
    })
    .returning();

  await tx.insert(analysisItems).values(
    items.map((item, index) => ({
      roastId: roast.id,
      severity: item.severity,
      title: item.title,
      description: item.description,
      order: index,
    }))
  );

  return roast;
});
```
